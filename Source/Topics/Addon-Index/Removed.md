# Removed Addons

The following addons have been removed from the Index for various reasons.

If you think of creating a new addon, have a look at this list, maybe you can 
rework, extend or maintain an existing project instead of starting from scratch.

<br/>

| Addon | Description |
|:------|:------------|
| [ExtMan] | Alternative to the Addon Manager - Succeeded by the FreeCAD Launcher
| [Autoload] | Automatically pre-loads workbenches - Integrated into Core
| [Flamingo] | Tools to draw frames and pipelines
| [Timber] | Tools for framing houses
| [Retr3d] | Creating 3D printers from e-waste

<br/>
