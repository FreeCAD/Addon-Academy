# Index

The [`Index.json`][Index] file is the heart of the Addon Index and contains the list of all registered addons.

To have your addon added to the Index, create an issue in the [GitHub FreeCAD/Addons repository][GitHubRepo], or alternately in the [Codeberg FreeCAD/Addons repository][CodebergRepo], or by posting in the [Addon Development Subforum][Forum] of the FreeCAD Forums. To be considered for addition, your Addon must meet the minimum quality standards outlined in [Qualities].
