# Publishing Your Addon

Various approaches to publishing your addon.

<br/>

## Approaches

### Portable

You can bundle your addon as a ZIP archive, share the file with others and have them manually install it.

### Reference

You can publish your addon's code in a repository and share the repository url with others that can use it to install your addon via the addon manager.

### Indexed

You can publish your addon's code in a repository and have it be added to the Addon Index which lists it in the addon manager.

<br/>

## Comparison

The `Portable` & `Reference` approaches are categorized as `Independent` as you have full control over how and where your addon is distributed.

The `Indexed` approach is the easiest way for users to find and install your addon, however you are required to follow certain rules and your addon to have certain qualities.

If you are unsure about which approach might suite you best, feel free to reach out. [» Contact][Contact]
